## Run Locally

- **Clone the project**

```bash
  git clone https://github.com/Sandip-Chavda/Real-Time-ChatApp.git
```

- **Install dependencies**

```bash
  npm install
```

- **Start the server**

```bash
  npm run dev
```

- Access the Application: Open your web browser and visit **`http://localhost:5173`** to access the **Real-Time Chat App**.
